# Commercial Real Estate Valuation Model

Python-based commercial real estate valuation and underwriting model built using the FY2024 Federal Real Property Profile (FRPP) dataset.

## Features

- Loads and processes 307,919 real estate records
- Filters office properties
- Generates rent roll assumptions
- Builds a 5-year pro forma
- Calculates NOI and leveraged cash flow
- Exports results to Excel

## Tech Stack

- Python
- Pandas
- NumPy
- OpenPyXL

## Dataset

Source: Federal Real Property Profile (FRPP FY2024)

Dataset Size:
- 307,919 rows
- 117 columns

## Output

- Asset Summary
- Rent Roll
- 5-Year Pro Forma
- Cash Flow Analysis

## Author

Portfolio project demonstrating commercial real estate valuation and financial modeling automation.
